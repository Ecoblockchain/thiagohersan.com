+ + + + =
iLikeYou (on facebook) is a web app designed to help us quantify our feelings towards each other. It goes through facebook data and tries to every post, picture and comment published by, or tagged with, a chosen friend.
Much like real love, the amount of data you can actually using this app is determined by the other person's privacy settings and how much they trust you.

This project is part of an ongoing series of web applications meant to explore different aspects of our technology-mediated interactions.
